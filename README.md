# vue2-vuex3-typescript

## Clone project
```
git clone https://github.com/Damjanose/vue2-vuex3-typescript.git
```
## Go at directory
```
cd ./vue2_typescript
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
